# Diagrams

This folder holds the visual assets referenced from the main `README.md` and the docs.

## Files to add here

| File | Purpose | Suggested source |
|---|---|---|
| `cover.png` | Hero image at the top of the README | The Gemini-generated cover image (server room + AWS icons + AI OPS AGENT plaque) |
| `architecture.png` | High-level pipeline diagram | Export the SVG flow diagram or hand-draw a sketchnote |
| `reasoning-loop.png` | The "2 vs 4 tool calls" branching visual | Optional but high-value — shows the agentic nature |

## Recommended dimensions

- **Cover image:** 1200 × 630 px (standard social-share / GitHub OG image size)
- **Architecture diagram:** 1600 px wide, transparent background if possible
- **Reasoning-loop visual:** 1200 px wide

## Tips

- Use PNG for raster images, SVG for diagrams that should scale
- Optimise file sizes — anything over 500 KB will slow GitHub page loads
- Add an `alt` text in the README for every image to keep the repo accessible
